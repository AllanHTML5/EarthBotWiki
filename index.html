About Earthbot

Earthbot is an automation tool (bot) that assists users by recording and repeating mouse movements based on visual triggers like color changes on screen. It's ideal for tasks involving repetitive scanning, selection, or rerolling.

It detects target stats based on a color rating system and acts accordingly to stop or repeat actions. Once configured, Earthbot can operate hands-free, with support for pause/resume hotkeys.

Target Match Example: You can configure Earthbot to stop when both Accuracy and Speed are GREEN, or more optimal (e.g., RED + PURPLE).

Installation

  1. Download the Earthbot ZIP file from the official link provided.
  2. Extract the contents to any folder on your PC.
  3. Double-click the executable file (Earthbot.exe) to launch.
  4. On first launch, input your provided license key and click Check.
  5. The bot will start faster on future launches.

Usage

  1. Record a Macro:
    • Click Record Macro.
    • Move your mouse into position when prompted.
    • Perform your desired actions.
    • Press ESC to finish recording.
  2. Define Target Regions:
    • Choose how many stats you want to check (e.g., 2 for Speed + Accuracy).
    • Click Select Regions.
    • Hover over an area and press R twice to define each region.
  3. Select Target Color: Choose your desired stat threshold color (e.g., Green or Red+Purple).
  4. Start Playback: Click Start and the bot will begin operating.
  5. Bot Control:
    • F12 — Pause or Resume the bot.
    • ESC — Stop the bot immediately.
  6. You can rerun Earthbot with the same settings without redefining macros or regions.

Tip: License and macro only need to be configured once unless the folder is deleted or reset.
